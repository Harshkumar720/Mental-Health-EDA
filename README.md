# Mental Health in Tech Industry - EDA

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on a mental health survey dataset to understand factors affecting employee mental health and treatment behavior.

## 🎯 Objective
To help organizations improve employee mental health by identifying key influencing factors such as family history, workplace support, and awareness.

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Key Insights
- Employees with family history are more likely to seek treatment
- Mental health affects work productivity significantly
- Employees are more comfortable discussing issues with coworkers than supervisors
- Availability of benefits increases treatment-seeking behavior

## 💡 Conclusion
Organizations should promote awareness, provide mental health benefits, and create a supportive environment to improve employee well-being.
